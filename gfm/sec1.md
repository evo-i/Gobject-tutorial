# Предварительные требования и лицензия

## Предварительные требования

### Документация учебника

Этот учебник посвящен библиотекам GObject.
Изначально они использовались на Linux с компилятором C, но теперь используются более широко: в Windows и macOS, с Vala, Python и т.д.
Однако этот учебник описывает только _программы на C для Linux_.

Если вы хотите скомпилировать примеры из учебника, вам потребуется:

- ПК с дистрибутивом Linux, таким как Ubuntu, Debian и т.д.
- Gcc
- GLib. Версия на момент написания этого документа — 2.76.1.
Некоторые примеры программ требуют версию 2.74 или выше.
Но они будут работать на более старых версиях, если вы замените новую функцию или макрос на старую.
- pkg-config
- meson и ninja

Обычные дистрибутивы Linux имеют GLib, которой достаточно для компиляции примеров из этого репозитория.

### Инструменты для создания файлов GFM, HTML и PDF

Этот репозиторий включает программы на Ruby.
Они используются для создания файлов Markdown (GFM), HTML, LaTeX и PDF.

Вам потребуется:

- Дистрибутив Linux, такой как Ubuntu.
- Язык программирования Ruby.
Существует два способа его установки.
Один — установка пакета дистрибутива.
Другой — использование rbenv и ruby-build.
Если вы хотите использовать последнюю версию Ruby, вам понадобятся rbenv и ruby-build.
- Rake.
Это gem — библиотека, написанная на Ruby.
Пакет Ruby включает Rake gem в качестве стандартной библиотеки, поэтому устанавливать его отдельно не нужно.
